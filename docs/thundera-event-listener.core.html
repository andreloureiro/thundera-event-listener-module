thundera-event-listener.core

A Node module to listen and register events created on Resolve applications

This module has only one public method, the registerEvent.

It receives an event type and the payload data for the event. Then, it validates against the events schemas registered in this module. If it’s all ok, the event data is sent to the Events API.

Author: Andre Loureiro

register-event

(register-event event-type data)

Receives the type of the event and the payload, converts the payload to EDN then validates it. If it passes validation, posts to the Events API to register it.

This method is public on the Node module and should be used like this:

var eventListenerModule = require(‘thundera-event-listener-module’);

eventListenerModule.registerEvent(‘CHANNEL_STATUS_CHANGE’, {…});
